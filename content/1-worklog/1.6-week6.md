## 3. Topics Covered

*   **Serverless Operations Automation:**
    *   Wrote Lambda automation scripts with IAM execution roles to discover and stop idle servers.
    *   Integrated Amazon EventBridge rules to trigger Lambda executions on fixed schedules.
    *   Connected an incoming webhook URL to post execution logs to a Slack channel.
*   **Custom Monitoring Dashboards (Grafana):**
    *   Launched a host VM, installed Grafana, and configured an IAM Instance Profile to query metrics.
    *   Built custom monitoring dashboards aggregating key metrics like CPU and disk I/O.
*   **Systems Manager Session Manager:**
    *   Wrote IAM Roles containing AWS-managed policies for Systems Manager.
    *   Established console-based shell sessions to eliminate standard SSH keys and port 22 vulnerabilities.
*   **AWS CloudFormation (IaC):**
    *   Provisioned resource configurations using JSON-based templates.
    *   Acquired understanding of the Stack lifecycle: creates, updates, drift detections, and deletion procedures.
*   **Personal Project API Gateway Integration:**
    *   Created an Amazon API Gateway REST API with a `/visitor` route pointing to the Lambda counter.
    *   Configured Cross-Origin Resource Sharing (CORS) rules to accept incoming web calls.
