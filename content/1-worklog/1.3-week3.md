## 3. Content

- Amazon EC2:
  - Created Windows and Linux EC2 instances, configured Security Groups, connected using RDP/SSH, and checked instance status.
  - Practiced EBS Snapshot, Custom AMI, and created a new instance from an AMI.
  - Configured IAM Role and used Systems Manager Session Manager to access EC2-Windows in case the Key Pair was lost.

- AWS Cloud9 / Kiro CLI:
  - Recorded that AWS Cloud9 is not available for new AWS accounts.
  - Installed, signed in, and used Kiro CLI in VS Code Terminal to create a sample Node.js project.

- Amazon Lightsail:
  - Deployed WordPress and PrestaShop on Amazon Lightsail.
  - Configured Lightsail Database, Static IP, and checked the websites through a browser.

- Auto Scaling:
  - Deployed an EC2 application model with Load Balancer and Auto Scaling.
  - Configured Launch Template, Target Group, ALB, and Auto Scaling Group.
  - Tested application access through the Load Balancer DNS.
