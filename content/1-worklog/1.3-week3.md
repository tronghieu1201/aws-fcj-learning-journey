## 3. Topics Covered

*   **Amazon EC2 Lifecycle & Virtualization:**
    *   Launched Linux instances (SSH-access) and Windows Server instances (RDP-access).
    *   Explored virtual storage blocks (EBS volumes), snapshot creations, and custom Amazon Machine Images (AMI).
    *   Implemented Systems Manager Session Manager for RDP connections as a secure backup access path.
*   **Kiro CLI Alternative IDE:**
    *   Adopted Kiro CLI (terminal-based workspace assistant) as AWS Cloud9 is restricted for newer student accounts.
    *   Successfully authenticated the client and initialized a sample Node.js environment.
*   **Amazon Lightsail Simplified Compute:**
    *   Launched a high-productivity single-node container/CMS instance for small projects.
    *   Configured separate managed databases, assigned static IPs, and customized billing alarms.
*   **Scalability & High Availability (ALB & ASG):**
    *   Created launch templates containing instance metadata and configurations.
    *   Set up an Application Load Balancer (ALB) pointing traffic to defined Target Groups.
    *   Configured an Auto Scaling Group (ASG) to dynamically maintain capacity based on traffic load.
