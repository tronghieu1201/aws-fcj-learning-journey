## 3. Content

- Amazon EC2: 
  - Practice creating EC2 Windows and Linux instances, configure Security Group, connect to instances using RDP/SSH and check server operating status.
  - EBS Snapshot and Custom AMI: Create Snapshot, create Custom AMI and launch a new instance from AMI to understand how to back up and reuse EC2 configuration.
  - Systems Manager: Configure IAM Role and use Session Manager to access EC2-Windows in case of losing the Key Pair.
- AWS Cloud9 / Kiro CLI:
    - Record that AWS Cloud9 is not available for new AWS accounts.
    - Install, log in and use Kiro CLI in VS Code Terminal to create a sample Node.js project.
- Application deployment: 
    - Deploy WordPress and PrestaShop on Amazon Lightsail.
    - Configure Lightsail Database, Static IP and test websites through the browser.
- Launch Template: 
  - Deploy an EC2 application model with Load Balancer and Auto Scaling.
  - Configure Launch Template, Target Group, ALB and Auto Scaling Group.
  - Test application access through the Load Balancer.
