## 3. Content

- AWS Backup:
  - Learned how to create a backup plan for AWS resources using AWS Backup.
  - Practiced configuring a backup plan, backup vault, and backup rule.
  - Became familiar with selecting resources for backup and checking backup jobs.

- SSL/HTTPS for the application:
  - Learned how to use Route 53, ACM, and CloudFront to configure HTTPS.
  - Practiced issuing an SSL certificate and attaching it to a CloudFront Distribution.
  - Configured website access through HTTPS to improve security.

- Docker and Amazon ECR:
  - Learned the basic concepts of Docker, images, and containers.
  - Practiced packaging an application into a Docker image.
  - Pushed the image to Amazon ECR to prepare for deployment on AWS.

- Amazon ECS:
  - Learned how to run containers on Amazon ECS.
  - Practiced creating an ECS Cluster, Task Definition, and Service.
  - Verified that the containerized application worked on ECS.

- Personal project - Visitor counter integration and production build:
  - Edited the personal portfolio source code using Vue/Vite.
  - Integrated the frontend with the API Gateway endpoint `/visitor`.
  - Checked that the visitor counter worked on the web interface.
  - Verified that visitor counter data was updated in DynamoDB.
  - Built the source code into a production version to prepare for upload to Amazon S3.
