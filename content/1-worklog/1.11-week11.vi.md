## 3. Nội dung

- Hoàn thiện sơ đồ kiến trúc AWS Cloud Portfolio:
  - Vẽ sơ đồ kiến trúc bằng draw.io.
  - Bỏ Route 53 khỏi sơ đồ vì project hiện tại chưa dùng custom domain.
  - Thể hiện đúng các thành phần đang dùng: User, CloudFront, S3, API Gateway, Lambda, DynamoDB, IAM Role và CloudWatch Logs.
  - Làm rõ luồng chính của project: người dùng truy cập website qua CloudFront, file frontend nằm trong S3, frontend gọi API Gateway để xử lý visitor counter.

- Kiểm thử lại website đã deploy trên AWS:
  - Mở website bằng CloudFront URL và kiểm tra website load bình thường.
  - Kiểm tra lại việc refresh trang không còn lỗi AccessDenied.
  - Kiểm tra S3 bucket đang chứa các file build frontend như `index.html`, `404.html` và thư mục `assets/`.
  - Xác nhận S3 chỉ chứa file build frontend, không phải source code. Source code vẫn được quản lý trên GitHub.

- Kiểm tra lại visitor counter:
  - Mở API Gateway endpoint `/visitor` trực tiếp trên trình duyệt.
  - Kiểm tra endpoint trả response JSON thành công.
  - Kiểm tra Lambda function `portfolioVisitorCounter` có logic xử lý visitor counter.
  - Kiểm tra Lambda dùng DynamoDB để cập nhật số lượt truy cập.
  - Kiểm tra CloudWatch Logs có ghi nhận quá trình Lambda chạy.

- Viết Workshop documentation cho project cá nhân:
  - Viết phần **Overview** để giới thiệu project AWS Cloud Portfolio và mục tiêu triển khai.
  - Viết phần **Prerequisites** để liệt kê các thành phần cần chuẩn bị như AWS account, GitHub, Node.js, S3, CloudFront, API Gateway, Lambda, DynamoDB và CloudWatch.
  - Viết phần **Architecture** để giải thích luồng CloudFront → S3 và API Gateway → Lambda → DynamoDB.
  - Viết phần **Deployment** để ghi lại cách build frontend, upload file build lên S3 và phân phối website bằng CloudFront.
  - Viết thêm phần Testing, Monitoring và Cleanup ở mức cơ bản để dùng cho báo cáo cuối.

- Rà soát lại nội dung Tuần 11:
  - Gom lại các ảnh minh chứng chính.
  - Đổi cách ghi từ “S3 bucket chứa source build frontend” thành “S3 bucket chứa file build frontend” cho đúng.
  - Kiểm tra lại các link chính của project:
    - GitHub source: `https://github.com/tronghieu1201/portfolio`
    - GitHub Pages backup: `https://tronghieu1201.github.io/portfolio/`
    - CloudFront live: `https://ddusufgoys11.cloudfront.net`
