## 3. Topics Covered

*   **Continuous Integration / Continuous Deployment:**
    *   Created multi-stage deployment pipelines using AWS CodePipeline.
    *   Configured AWS CodeBuild stages to build, test, and package serverless code automatically on git push events.
*   **Event-Driven Serverless Workloads:**
    *   Connected S3 object creation triggers (`ObjectCreated:Put`) to trigger a downstream Lambda function.
    *   Wrote data-parsing scripts to store file metadata inside Amazon DynamoDB.
*   **AWS Serverless Application Model (SAM):**
    *   Used AWS SAM templates to model serverless applications (APIs, Lambdas, and Databases) as declarative YAML.
    *   Built and deployed SAM architectures using the `sam build` and `sam deploy` CLI commands.
*   **Personal Project CDN Hosting Integration:**
    *   Uploaded the optimized static build directory (`dist/`) to the portfolio S3 bucket.
    *   Created an Amazon CloudFront distribution pointing to the S3 bucket to cache portfolio assets globally.
    *   Verified global secure HTTPS access via the CloudFront domain:
        *   `https://ddusufgoys11.cloudfront.net`
