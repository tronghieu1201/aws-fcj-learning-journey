## 3. Content

- CI/CD for a serverless application:
  - Learned about the automated build and deployment process for serverless applications.
  - Became familiar with CodePipeline, CodeBuild, and deployment automation.
  - Practiced configuring a pipeline to reduce manual deployment steps.

- Lambda, S3, and DynamoDB:
  - Learned how AWS Lambda handles backend logic in a serverless architecture.
  - Practiced integrating Lambda with S3 and DynamoDB.
  - Became familiar with storing data and processing events without managing servers.

- API Gateway and Frontend:
  - Learned how to create an API endpoint using Amazon API Gateway.
  - Connected the frontend with API Gateway to invoke a Lambda function.
  - Configured CORS so that the frontend could call the API correctly.

- AWS SAM:
  - Learned how to use AWS SAM to deploy a serverless application using a template.
  - Practiced defining Lambda, API Gateway, and DynamoDB through a configuration file.
  - Deployed a serverless application through SAM instead of manually creating each service.

- Personal project - Deployment to S3 and CloudFront:
  - Built the portfolio source code into the `dist` folder.
  - Checked the build files, including `index.html`, `assets/`, and other required static files.
  - Uploaded the production build to Amazon S3.
  - Created a CloudFront Distribution pointing to the S3 bucket.
  - Accessed the website through the CloudFront default domain:
    - `https://ddusufgoys11.cloudfront.net`
  - Verified that the website loaded successfully through CloudFront default HTTPS.
