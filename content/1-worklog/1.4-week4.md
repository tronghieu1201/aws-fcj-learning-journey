## 3. Content

- Host a static website with Amazon S3:
  - Practiced creating an S3 bucket, uploading source code, and enabling Static Website Hosting.
  - Configured access permissions so the website could be accessed through a browser.
  - Learned the role of CloudFront in caching content and improving website loading speed.

- Amazon RDS:
  - Learned the role of relational databases in application architecture on AWS.
  - Practiced creating a database with Amazon RDS and became familiar with DB instance, engine, security group, and endpoint.
  - Created an RDS MySQL database in a private subnet and configured a DB Subnet Group.
  - Configured Security Group rules so EC2 could connect to RDS through port 3306.
  - Deployed a Node.js application on EC2 and configured the `.env` file to connect to the RDS endpoint.
  - Imported sample data from `users.sql` into the database and verified that the web app displayed data from RDS.

- Amazon CloudWatch:
  - Learned the role of CloudWatch in monitoring AWS resources.
  - Practiced monitoring metrics and logs, and created basic alarms for the system.

- Starting the Personal Project - AWS Cloud Portfolio:
  - Designed portfolio page content, including self-introduction, skills, and AWS learning timeline.
  - Built the portfolio page using HTML/CSS, with possible support from an AI coding assistant.
  - Uploaded the first version to an S3 bucket and accessed it through the website endpoint.
  - This is the initial foundation and will be improved with Lambda, API Gateway, DynamoDB, and CloudFront in the following weeks.
