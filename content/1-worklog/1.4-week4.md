## 3. Topics Covered

*   **Amazon S3 Static Hosting:**
    *   Created S3 buckets, disabled block public access options, and configured bucket policies.
    *   Uploaded assets and verified global access via the static website hosting endpoint.
*   **Amazon RDS Private Databases:**
    *   Launched a managed MySQL RDS instance in private subnets using custom DB Subnet Groups.
    *   Configured port 3306 Security Group policies to restrict database access only to authorized EC2 instances.
    *   Deployed a backend Node.js server on EC2 and connected it to the database using an `.env` config file containing the RDS connection endpoint.
*   **Amazon CloudWatch Auditing:**
    *   Monitored instance metrics and configured Simple Notification Service (SNS) alarms to send email alerts on high CPU usage.
*   **Personal Project Initiation (AWS Cloud Portfolio):**
    *   Designed and structured an initial HTML/CSS portfolio template.
    *   Uploaded the layout to S3 to prepare for future serverless backend integrations.
