## 3. Nội dung

- Host website tĩnh với Amazon S3:
  - Thực hành tạo S3 bucket, upload source code và bật Static Website Hosting.
  - Cấu hình quyền truy cập để website có thể truy cập qua trình duyệt.
  - Tìm hiểu vai trò của CloudFront trong việc cache nội dung và tăng tốc độ tải trang.

- Amazon RDS:
  - Tìm hiểu vai trò của database quan hệ trong kiến trúc ứng dụng trên AWS.
  - Thực hành tạo database với Amazon RDS và làm quen với DB instance, engine, security group và endpoint.
  - Tạo RDS MySQL trong private subnet và cấu hình DB Subnet Group.
  - Cấu hình Security Group để EC2 kết nối tới RDS qua port 3306.
  - Deploy ứng dụng Node.js trên EC2 và cấu hình .env kết nối tới RDS endpoint.
  - Import dữ liệu mẫu từ users.sql vào database và kiểm tra web app hiển thị dữ liệu từ RDS.
  
- Amazon CloudWatch:
  - Tìm hiểu vai trò của CloudWatch trong việc giám sát tài nguyên AWS.
  - Thực hành theo dõi metrics, logs và tạo alarm cơ bản cho hệ thống.

- Khởi động Project cá nhân - AWS Cloud Portfolio:
  - Thiết kế nội dung trang portfolio: giới thiệu bản thân, skills, timeline học AWS.
  - Build HTML/CSS cho trang portfolio, có thể sử dụng AI coding assistant để hỗ trợ.
  - Upload phiên bản đầu tiên lên S3 bucket và truy cập qua endpoint.
  - Đây là nền tảng ban đầu, sẽ được bổ sung thêm Lambda, API Gateway, DynamoDB và CloudFront ở các tuần tiếp theo.
