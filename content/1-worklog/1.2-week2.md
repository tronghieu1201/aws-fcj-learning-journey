## 3. Topics Covered

*   **Amazon VPC Networking:**
    *   Explored fundamental networking blocks: VPCs, subnets, route tables, and Internet Gateways (IGW).
    *   Provisioned a custom VPC, enabled DNS resolution, and enabled DNS hostnames.
    *   Constructed routing tables to connect public subnets with the Internet Gateway.
    *   Used the VPC Resource Map visualization tool to inspect topology.
*   **IAM Roles for Compute:**
    *   Analyzed advantages of utilizing IAM Roles over hard-coding static Access Keys within EC2.
    *   Provisioned an IAM Instance Profile granting S3 read access, and successfully attached it to an running EC2 instance.
*   **Amazon Route 53 Private Resolver:**
    *   Studied Hybrid DNS architectures connecting on-premises servers with Cloud infrastructure.
    *   Deployed lab resources using CloudFormation templates.
    *   Provisioned Route 53 Inbound and Outbound Endpoints and defined custom resolver rules.
    *   Validated correct bi-directional name resolution via `nslookup` command tests.
*   **AWS CLI Administration:**
    *   Learned installation, profile setup, and interactive querying in terminal environments.
    *   Verified target instance permissions fetched from the metadata service.
    *   Executed basic AWS CLI operations on S3, SNS, IAM, and EC2, followed by clean-up scripts.
