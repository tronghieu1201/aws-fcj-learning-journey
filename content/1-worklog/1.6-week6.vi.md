## 3. Nội dung

- AWS Lambda và Slack:
  - Tìm hiểu cách sử dụng AWS Lambda để tự động hóa thao tác vận hành EC2.
  - Thực hành tạo IAM Role cấp quyền cho Lambda thao tác với EC2 và CloudWatch Logs.
  - Gắn tag cho EC2 để Lambda nhận diện đúng instance cần bật/tắt.
  - Cấu hình Lambda function để tự động start/stop EC2 theo nhu cầu sử dụng.
  - Tích hợp Slack Webhook để gửi thông báo khi Lambda thực hiện thao tác thành công.

- CloudWatch và Grafana:
  - Tìm hiểu cách giám sát tài nguyên AWS thông qua CloudWatch metrics.
  - Thực hành tạo EC2 và cấu hình IAM Role để Grafana có quyền đọc dữ liệu giám sát.
  - Cài đặt Grafana trên EC2 và kết nối Grafana với CloudWatch làm data source.
  - Tạo dashboard cơ bản để theo dõi CPU, network và trạng thái tài nguyên.
  - Làm quen với việc trực quan hóa dữ liệu giám sát thay vì chỉ xem metrics trực tiếp trên AWS Console.

- AWS Systems Manager Session Manager:
  - Tìm hiểu cách truy cập EC2 an toàn bằng Session Manager mà không cần SSH key.
  - Thực hành cấu hình IAM Role cho EC2 để sử dụng AWS Systems Manager.
  - Kết nối vào EC2 public/private thông qua Session Manager trên AWS Console.
  - Hiểu lợi ích của Session Manager trong việc giảm rủi ro bảo mật do không cần mở port SSH 22.
  - Kiểm tra session logs để theo dõi lịch sử truy cập và thao tác trên instance.

- AWS CloudFormation:
  - Tìm hiểu khái niệm Infrastructure as Code trong triển khai hạ tầng AWS.
  - Thực hành viết và sử dụng CloudFormation template để tạo tài nguyên tự động.
  - Tạo CloudFormation Stack, kiểm tra trạng thái triển khai và xem danh sách tài nguyên được tạo.
  - Cập nhật hoặc xóa stack để quản lý vòng đời tài nguyên.
  - Hiểu cách CloudFormation giúp triển khai hạ tầng nhất quán, dễ lặp lại và dễ quản lý hơn thao tác thủ công.
