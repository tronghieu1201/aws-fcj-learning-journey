## 3. Nội dung

- IAM Permission Boundary:
  - Tìm hiểu cách giới hạn quyền tối đa của IAM User bằng Permission Boundary.
  - Thực hành tạo policy giới hạn, tạo IAM User và kiểm tra quyền thực tế của user.
- IAM Role & Condition:
  - Ôn lại IAM User, Group, Role và quá trình Assume Role.
  - Thực hành tạo role và giới hạn quyền truy cập theo điều kiện như IP hoặc thời gian.
- AWS Security Hub:
  - Kích hoạt Security Hub để kiểm tra trạng thái bảo mật tài khoản AWS.
  - Xem security standards, findings và điểm đánh giá bảo mật theo benchmark.
- AWS WAF:
  - Tìm hiểu cách bảo vệ ứng dụng web bằng Web ACL.
  - Thực hành cấu hình managed rules, custom rule và kiểm thử request bị chặn/cho phép.
