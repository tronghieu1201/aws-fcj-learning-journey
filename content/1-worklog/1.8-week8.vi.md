## 3. Nội dung

- **AWS Backup:**
  - Tìm hiểu cách tạo kế hoạch sao lưu tài nguyên AWS bằng AWS Backup.
  - Thực hành cấu hình backup plan, backup vault và rule sao lưu.
  - Làm quen với việc chọn tài nguyên cần backup và kiểm tra backup job.
- **SSL/HTTPS cho ứng dụng:**
  - Tìm hiểu cách dùng Route 53, ACM và CloudFront để cấu hình HTTPS.
  - Thực hành cấp SSL certificate và gắn vào CloudFront Distribution.
  - Cấu hình truy cập website qua HTTPS để tăng bảo mật.
- **Docker và Amazon ECR:**
  - Tìm hiểu cơ bản về Docker, image và container.
  - Thực hành đóng gói ứng dụng thành Docker image.
  - Đẩy image lên Amazon ECR để chuẩn bị triển khai trên AWS.
- **Amazon ECS:**
  - Tìm hiểu cách chạy container trên Amazon ECS.
  - Thực hành tạo ECS Cluster, Task Definition và Service.
  - Kiểm tra ứng dụng container hoạt động trên ECS.
- **Project cá nhân - Tích hợp visitor counter và build production:**
  - Chỉnh sửa source portfolio cá nhân bằng Vue/Vite.
  - Tích hợp frontend with API Gateway endpoint `/visitor`.
  - Kiểm tra visitor counter hoạt động trên giao diện web.
  - Kiểm tra dữ liệu visitor counter được cập nhật trong DynamoDB.
  - Build source thành bản production để chuẩn bị upload lên Amazon S3.
