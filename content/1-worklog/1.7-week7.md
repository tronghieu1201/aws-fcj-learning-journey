## 3. Content

- IAM Permission Boundary:
  - Learned how to limit the maximum permissions of an IAM User using a Permission Boundary.
  - Practiced creating a boundary policy, creating an IAM User, and testing the user’s actual permissions.

- IAM Role and Condition:
  - Reviewed IAM User, Group, Role, and the Assume Role process.
  - Practiced creating a role and restricting access based on conditions such as IP address or time.

- AWS Security Hub:
  - Enabled Security Hub to check the security posture of the AWS account.
  - Reviewed security standards, findings, and security benchmark scores.

- AWS WAF:
  - Learned how to protect a web application using a Web ACL.
  - Practiced configuring managed rules, custom rules, and testing blocked or allowed requests.
