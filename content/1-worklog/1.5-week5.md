## 3. Topics Covered

*   **S3 Secure Bucket Topologies:**
    *   Built test architectures using parameterized CloudFormation templates (modified EC2 sizes to `t3.micro` to maintain Free Tier limits).
    *   Configured separate AWS CLI profiles (`user1`, `user2`) to test cross-boundary bucket accesses.
    *   Wrote S3 Bucket Policies restricting HTTP access, accepting only TLS/HTTPS data transfers.
*   **IAM Policy Governance:**
    *   Learned to prevent cost overruns using declarative IAM policies.
    *   Configured IAM Policies using conditional operators (`StringEquals`, `ArnEquals`) that prevent spinning up high-cost server types, unauthorized EBS models, or working in unauthorized AWS Regions.
*   **Tagging Strategy & Resource Groups:**
    *   Managed metadata boundaries on AWS using logical Key-Value tags.
    *   Organized distributed resources into visual units using Tag-Based Resource Groups.
*   **Personal Project Serverless Counter backend:**
    *   Created an Amazon DynamoDB NoSQL table (`cloud-portfolio-table`) using a partition key (`id`) to track site page visits.
    *   Wrote a Node.js-based AWS Lambda function to increment and return database visitor logs.
    *   Assigned a custom IAM execution role allowing the Lambda function to perform read/write queries on DynamoDB.
