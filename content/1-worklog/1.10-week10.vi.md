## 3. Nội dung

- **Amazon Cognito:**
  - Tìm hiểu cách xác thực người dùng trong ứng dụng serverless.
  - Thực hành tạo User Pool và cấu hình đăng ký, đăng nhập.
  - Làm quen với token xác thực và cách bảo vệ API.
- **SQS và SNS:**
  - Tìm hiểu cách xử lý tác vụ bất đồng bộ bằng hàng đợi và thông báo.
  - Thực hành dùng SQS để lưu message chờ xử lý.
  - Tìm hiểu SNS để gửi thông báo đến nhiều nơi nhận.
  - Kết hợp Lambda với SQS/SNS để xử lý luồng sự kiện.
- **CloudWatch và X-Ray:**
  - Tìm hiểu cách giám sát ứng dụng serverless trong môi trường production.
  - Thực hành xem logs, metrics và lỗi phát sinh trong CloudWatch.
  - Làm quen với AWS X-Ray để phân tích luồng request và thời gian xử lý.
- **AWS Step Functions:**
  - Tìm hiểu cách điều phối workflow nhiều bước trên AWS.
  - Thực hành tạo state machine để kết nối nhiều bước xử lý.
  - Hiểu cách Step Functions giúp quản lý quy trình phức tạp rõ ràng hơn.
- **Project cá nhân - Hoàn thiện source và CloudFront default domain:**
  - Rà soát lại source code portfolio trên GitHub.
  - Cập nhật README.md để ghi đúng tên project, link source, link GitHub Pages backup và link CloudFront live chính.
  - Kiểm tra lại bản build production trước khi upload lên S3.
  - Kiểm tra CloudFront Distribution đang ở trạng thái Enabled.
  - Kiểm tra website chạy bằng CloudFront default domain: `https://ddusufgoys11.cloudfront.net`
  - Xử lý lỗi refresh trang chi tiết project bị `AccessDenied` bằng cấu hình CloudFront Error Pages.
  - Kiểm tra modal xác nhận project hoạt động đúng.
  - Chuẩn hóa nội dung project để chuẩn bị sang Tuần 11 vẽ sơ đồ kiến trúc và viết tài liệu hướng dẫn.
