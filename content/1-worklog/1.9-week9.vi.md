## 3. Nội dung

- CI/CD cho ứng dụng Serverless:
  - Tìm hiểu quy trình build và deploy ứng dụng serverless tự động.
  - Làm quen với CodePipeline, CodeBuild và cách tự động hóa triển khai.
  - Thực hành cấu hình pipeline để giảm thao tác deploy thủ công.

- Lambda, S3 và DynamoDB:
  - Tìm hiểu cách AWS Lambda xử lý logic backend trong mô hình serverless.
  - Thực hành Lambda tương tác với S3 và DynamoDB.
  - Làm quen với cách lưu trữ dữ liệu và xử lý sự kiện không cần quản lý server.

- API Gateway và Frontend:
  - Tìm hiểu cách tạo API endpoint bằng Amazon API Gateway.
  - Kết nối frontend với API Gateway để gọi Lambda function.
  - Cấu hình CORS để frontend có thể gọi API hợp lệ.

- AWS SAM:
  - Tìm hiểu cách dùng AWS SAM để triển khai ứng dụng serverless bằng template.
  - Thực hành khai báo Lambda, API Gateway và DynamoDB bằng file cấu hình.
  - Deploy ứng dụng serverless thông qua SAM thay vì tạo thủ công từng dịch vụ.

- Project cá nhân - Deploy lên S3 và CloudFront:
  - Build source portfolio thành thư mục `dist`.
  - Kiểm tra các file build gồm `index.html`, `assets/` và các file tĩnh cần thiết.
  - Upload bản build lên Amazon S3.
  - Tạo CloudFront Distribution trỏ về S3 bucket.
  - Truy cập website bằng CloudFront default domain:
    - `https://ddusufgoys11.cloudfront.net`
  - Kiểm tra website load thành công qua HTTPS mặc định của CloudFront.
