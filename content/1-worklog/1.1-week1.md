## 3. Content

- AWS Account:
  - Created a new AWS account and successfully logged in to the AWS Management Console.
  - Checked account information, Free Plan status, credits, and became familiar with the Console Home interface.

- AWS Budgets:
  - Learned how to monitor AWS usage costs.
  - Created a budget and configured cost alerts to avoid unexpected charges.

- AWS Support:
  - Accessed the AWS Support Center to explore the support area.
  - Checked the current support plan of the account and reviewed the list of support cases.

- AWS IAM:
  - Learned about IAM Users, Groups, Roles, and Policies.
  - Understood the basic role of IAM in managing AWS service access permissions based on security best practices.
