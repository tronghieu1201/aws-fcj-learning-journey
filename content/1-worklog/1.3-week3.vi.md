## 3. Nội dung

- Amazon EC2: 
  - Thực hành tạo EC2 Windows và Linux, cấu hình Security Group, kết nối vào instance bằng RDP/SSH và kiểm tra trạng thái hoạt động của máy chủ.
  - EBS Snapshot và Custom AMI: Tạo Snapshot, tạo Custom AMI và khởi tạo instance mới từ AMI để hiểu cách sao lưu và tái sử dụng cấu hình EC2.
  - Systems Manager: Cấu hình IAM Role và sử dụng Session Manager để truy cập EC2-Windows trong tình huống mất Key Pair.
- AWS Cloud9 / Kiro CLI:
    - Ghi nhận AWS Cloud9 không khả dụng với tài khoản AWS mới.
    - Cài đặt, đăng nhập và sử dụng Kiro CLI trong VS Code Terminal để tạo project Node.js mẫu.
- Triển khai ứng dụng : 
    - Triển khai WordPress và PrestaShop trên Amazon Lightsail.
    - Cấu hình Lightsail Database, Static IP và kiểm tra website qua trình duyệt.
- Launch Template : 
  - Triển khai mô hình ứng dụng EC2 có Load Balancer và Auto Scaling.
  - Cấu hình Launch Template, Target Group, ALB và Auto Scaling Group.
  - Kiểm thử truy cập ứng dụng thông qua Load Balancer.
