## 3. Topics Covered

*   **User Identity Management (Cognito):**
    *   Created Cognito User Pools to securely handle registration, sign-in, and account confirmation.
    *   Integrated JWT (JSON Web Tokens) validation to secure backend REST endpoints.
*   **Asynchronous Message Processing:**
    *   Built decoupled, fault-tolerant messaging channels using SQS (Simple Queue Service) and SNS (Simple Notification Service).
    *   Implemented fan-out event architectures to trigger parallel backend processes.
*   **Distributed App Performance (AWS X-Ray):**
    *   Wrote X-Ray instrumentation libraries to trace API latencies and downstream database request times.
*   **Process Coordination (AWS Step Functions):**
    *   Orchestrated complex, multi-step business logic using ASL (Amazon States Language) State Machines.
*   **Personal Project SPA Route Fixes:**
    *   Solved the common Single Page Application (SPA) refresh issue where reloading nested routes returned `AccessDenied` errors from S3.
    *   Configured Custom Error Pages on the CloudFront distribution to redirect 403/404 response codes to `/index.html` with a 200 OK status.
    *   Configured a GitHub Actions workflow to automate building and deploying the Vue client to S3 on git pushes.
