## 3. Nội dung

- **Mạng AWS:**
  - Tìm hiểu các thành phần mạng cơ bản trong AWS gồm Amazon VPC, subnet, route table và Internet Gateway.
  - Thực hành tạo VPC, bật DNS resolution và DNS hostnames.
  - Cấu hình subnet, route table và Internet Gateway để xây dựng mô hình mạng cơ bản.
  - Sử dụng VPC Resource Map để quan sát tổng quan tài nguyên mạng.
- **IAM Role:**
  - Tìm hiểu cách cấp quyền cho EC2 truy cập dịch vụ AWS bằng IAM Role.
  - So sánh cách sử dụng Access Key trực tiếp với cách sử dụng IAM Role.
  - Thực hành tạo IAM Role, gán policy S3 và attach role vào EC2 instance.
- **Route 53:**
  - Tìm hiểu mô hình Hybrid DNS giữa môi trường AWS và hệ thống DNS nội bộ/on-premise.
  - Khởi tạo hạ tầng lab bằng CloudFormation.
  - Tạo Route 53 Resolver Outbound Endpoint, Inbound Endpoint và Resolver Rule.
  - Kiểm tra phân giải DNS bằng `nslookup` và kiểm tra kết nối bằng `ping`.
- **AWS CLI:**
  - Tìm hiểu cách sử dụng AWS CLI để thao tác với dịch vụ AWS bằng dòng lệnh.
  - Kiểm tra AWS CLI trên EC2 và xác minh credentials được lấy từ IAM Role.
  - Thao tác với S3, SNS, IAM, VPC và EC2 bằng command line.
  - Dọn dẹp tài nguyên sau khi hoàn thành lab.
