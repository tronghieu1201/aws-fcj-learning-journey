3. Topics Covered

  • AWS Backup Disaster Recovery:
    • Configured automated snapshot schedules and lifecycle policies using centralized Backup Plans.
    • Created secure Backup Vaults to store system-wide backups.
  • SSL/TLS ACM Integration:
    • Requested public SSL certificates using AWS Certificate Manager (ACM).
    • Configured secure HTTPS redirection rules on an Application Load Balancer.
  • Docker Containerization:
    • Wrote multi-stage Dockerfiles to optimize image footprint.
    • Created secure private repositories in Amazon Elastic Container Registry (ECR) and pushed container images using the docker CLI.
  • Amazon ECS (Elastic Container Service):
    • Explored container orchestration models, focusing on Task Definitions, Services, and Clusters.
    • Deployed containerized applications on serverless AWS Fargate compute.
  • Personal Project Vue/Vite Build:
    • Modified the portfolio UI source code to send HTTP requests to the API Gateway /visitor endpoint.
    • Verified real-time counter updates on the Vue frontend.
    • Ran a production build with Vite to output the optimized static build directory (dist/).

4. Key Takeaways

  • Automated backup schedules for disaster recovery.
  • Configured end-to-end TLS/SSL encryption.
  • Packaged web applications into Docker container images.
  • Deployed containerized applications on AWS Fargate.
  • Compiled the Vue frontend to optimized static files.
