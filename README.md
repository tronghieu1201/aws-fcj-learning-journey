## About This Project

This is my personal internship report website built with [Hugo](https://gohugo.io/) and the [hugo-theme-learn](https://github.com/matcornic/hugo-theme-learn) theme, based on the [FCJ Workshop Template](https://github.com/thienluhoan/fcj-workshop-template).

## Structure

```
content/
├── _index.md              # Homepage
├── 1-worklog/             # Weekly worklog (12 weeks)
├── 2-proposal/            # Internship proposal
├── 3-blogsposted/         # Published blog posts
├── 4-eventparticipated/   # Events participated
├── 5-workshop/            # Original workshop
├── 6-self-evaluation/     # Self-assessment
└── 7-feedback/            # Sharing and feedback
```

## How to Run Locally

```bash
# Clone the repo with submodules
git clone --recurse-submodules https://github.com/YOUR_USERNAME/fcj-workshop-hieu.git

# Navigate to the project
cd fcj-workshop-hieu

# Run Hugo server
hugo server
```

Then open http://localhost:1313 in your browser.
